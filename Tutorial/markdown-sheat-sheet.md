# Markdown Cheat Sheet

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Here's a quick reference guide:

## Headers

```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Emphasis

```markdown
*italic* or _italic_
**bold** or __bold__
***bold and italic***
```

## Lists

**Unordered List:**

```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

**Ordered List:**

```markdown
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
```

## Links

```markdown
[Link Text](http://www.example.com)
```

## Images

```markdown
![Alt Text](image.jpg)
```

## Blockquotes

```markdown
> This is a blockquote.
```

## Code

**Inline Code:**

```markdown
`code`
```

**Code Blocks:**

```markdown
```python
def hello_world():
    print("Hello, World!")
```

```

## Horizontal Rule

```markdown
---
```

## Tables

```markdown
| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |
```

## Line Break

```markdown
Text before  
a line break.
```

## Escaping Characters

```markdown
\*This is not italic\*
```

This is just a basic cheat sheet. Markdown supports more features and customization. Refer to the [official Markdown documentation](https://www.markdownguide.org/) for a comprehensive guide.

```

Feel free to customize it further based on your needs or add more advanced features as required.
